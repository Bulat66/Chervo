Технология комплексной переработки птичьего помета предназначена для применения в сельском хозяйстве, а конкретно в птицеводстве. С помощью элекрофлотокоагуляции будет выделяться 97% белка, содержащегося в помете, который после сушки и гранулирования может быть использован в кормах для животных и для других целей. На основе водного экстракта будет получено жидкое органическое удобрение содержащее макро- и микроэлементы.

Из нерастворимых компонентов с применением микробиологических препаратов по ускоренной технологии будут получены твёрдые органические удобрения, которые рекомендуются для структурирования тяжёлых почв и выступают в качестве органического удобрения пролонгированного действия. Используя предлагаемую технологию, птицефабрики снизят расходы на производство птицеводческой продукции (снизится себестоимость мяса бройлеров на 5%), расширят ассортимент выпускаемой продукции, уменьшат количество штрафов выплачиваемых за нарушения экологического законодательства. Предлагаемая технология позволит проводить постоянную переработку свежего помёта бройлеров.

Целевая аудитория: Разработка будет применяться в сфере сельского хозяйства. В России насчитывается 615 птицеводческих предприятий 165 из них Бройлерные птицефабрики, которые в будущем могут стать нашими потенциальными клиентами, с которыми заключатся договора на инженерное решение комплексной переработки куриного помета. Будет проводиться оценка состояния птицефабрики (количество помета в день, наличие площадок для монтажа оборудования, состояния коммуникаций и др.). В дальнейшем с птицефабрикой заключается договор на обслуживание оборудования и поставку необходимых реагентов. Среди всех работающих предприятий по переработке помета кур и его утилизации не одно из них не выделяет белок из помета. В нашей разработке производится получение удобрений из птичьего помёта с высоким содержанием азота, фосфора и калия. В процессе переработки не обнаруживается наличие патогенных бактерий. Запах при производстве отсутствует.

  • Нашими целевыми продуктами являются:
  • Белковый корм
  • Жидкое органическое удобрение
  • Твердое органическое удобрение, улучшающее структуру почвы

Данная технология является безотходной, так как водный экстракт после стандартизации может быть использован как жидкое органическое удобрение, а нерастворимые компоненты после микробиологической обработки могут быть использованы в качестве твердых органических удобрений. С использованием этой технологии сокращается время переработки куриного помета в 3-4 раза по сравнению с традиционными методами. Внедрение данной технологии позволит в дальнейшем птицефабрикам получать дополнительные доходы за счет производимых продуктов (белок, жидкие и твердые органические удобрения).
